# 创建你的角色

角色是存储在系统的哈希表内的，由系统端发送控制信息，通过渲染端进行管理
所有角色的接口都存于注入的Character表内，lib文件夹已经有了详细说明。

创建实际上相当简单,仅仅需要一行

```lua
Character.createNewCharacter(url,id)
```
| 参数  | 类型     | 说明       |
| --- | ------ | -------- |
| url | string | 模型的文件夹路径 |
| id  | string | 角色的唯一标识符 |


当然，删除也不难

```lua
Character.deleteCharacter(id)
```
| 参数 | 类型     | 说明       |
| -- | ------ | -------- |
| id | string | 角色的唯一标识符 |


由此可见，角色的唯一标识符是控制角色的根本依据，在本章后续您将看到更多角色接口，而它们都会与唯一标识符有关联
